1. In Solution Explorer, right-click the name of the project that you want to add the service to, and then click Add Service Reference.
2. In the Address box, enter the URL for the service, and then click Go to search for the service.
3. In the Namespace box, enter the namespace that you want to use for the reference. ("FileUploadSvcRef" in this case)
4. Click OK to add the reference to the project.
